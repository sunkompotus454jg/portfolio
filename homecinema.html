HomeCinema

Собственный локальный домашний кинотеатр, связывающий Android TV с сетевым хранилищем.

Платформа: Android TV / Web
Технологии:
Android TVMedia ServerWeb Admin Panel

Концепция

HomeCinema — это проект, который связывает телевизоры на базе Android с локальным хранилищем (NAS) в домашней сети, позволяя с комфортом смотреть фильмы и сериалы.

Оформление и пользовательский опыт (UX) вдохновлены популярными стриминговыми сервисами, но работают полностью локально и без задержек.

Ключевые фичи (Текущий билд)

  • Собственная Админ-панель: Удобный веб-интерфейс, через который легко загружать новые фильмы в библиотеку.
  • Умное автозаполнение: При добавлении фильма обложки, описания и метаданные подтягиваются автоматически.
  • Плавный UI для Android TV: Интерфейс адаптирован специально для управления с пульта ДУ телевизора.

Проект на данный момент не опубликован на GitHub, так как находится в стадии активной разработки, но уже имеет полностью рабочий билд.

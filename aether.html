Aether (Amodule)

Десктопный AI-ассистент для студентов — полупрозрачный overlay поверх всех окон с захватом аудио и экрана.

Платформа: Windows 10/11
Технологии:
ElectronVanilla JSGroq LLMGemini Flash

Возможности

  • Захват аудио: запись с микрофона + транскрипция через Whisper (Groq API).
  • OCR экрана: скриншот выбранной области + распознавание текста через Gemini Vision.
  • LLM ответы: быстрая генерация кратких, точных ответов через Groq LLM.
  • Невидимость: overlay полностью невидим при демонстрации экрана (Windows 10/11).
  • Горячие клавиши: управление без отвлечения от основного процесса.

Установка

# Клонирование
git clone https://github.com/sunkompotus454jg/Aether.git
cd Aether

# Установка зависимостей
npm install

# Запуск
npm start

Горячие клавиши

КомбинацияДействие
Ctrl+Shift+HСкрыть/показать overlay
Ctrl+Shift+AВключить/выключить запись аудио
Ctrl+Shift+SСделать скриншот области
Ctrl+Shift+XПереключить невидимость для демонстрации экрана
Ctrl+Shift+QВыход из приложения

API Ключи

Для работы необходимы бесплатные API-ключи:

  • Groq API — console.groq.com (STT + LLM, бесплатно)
  • Google Gemini API — aistudio.google.com (OCR, бесплатно)

Ключи вводятся в настройках приложения (иконка ).

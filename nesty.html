Nesty

Стильная и легковесная утилита для Windows, созданная для идеального порядка на рабочем столе.

Платформа: Windows Desktop
Технологии:
PythonPyQt6Windows API

Главные фишки

  • Умное сворачивание: Окна автоматически сворачиваются в тонкую шапку, когда мышь уходит за их пределы, и плавно разворачиваются при наведении.
  • Магнитное прилипание (Snapping): Сетки аккуратно магнитятся друг к другу и к краям экрана (дистанция 20 пикселей) для создания идеального пиксельного порядка.
  • Глубокая кастомизация: 5 встроенных неоновых пресетов (Синий, Фиолетовый, Зеленый, Оранжевый, Красный) + возможность создать свой собственный пресет по HEX-коду через встроенную палитру.
  • Безопасность файлов: При удалении сетки все файлы из неё автоматически и безопасно перемещаются обратно на Рабочий стол.
  • Интеграция с Windows: Программа добавляет пункт "Создать Сетку (Fences)" прямо в контекстное меню рабочего стола и автоматически запускается вместе с системой.
  • Drag & Drop: Файлы легко перетаскиваются внутрь сетки обычным движением мыши.

Установка

Поскольку программа упакована в полноценный инсталлятор, установка занимает всего пару кликов:

  1. Запустите файл Nesty_Setup.exe.
  2. Следуйте инструкциям на экране. Установщик автоматически добавит программу в Автозагрузку и пропишет контекстное меню.
  3. Готово! Программа уже работает в фоновом режиме.

Для разработчиков (Сборка из исходников)

pip install pyqt6 pyinstaller
pyinstaller --noconsole --onefile --name="Nesty" --icon=icoc.ico main.py

DeckBoard

Мощная экосистема для дистанционного управления вашим компьютером прямо со смартфона.

Платформа: Android + Windows
Технологии:
Kotlin/AndroidPython Flask

Основные возможности

Android Приложение (Client)

  • Кастомизация: Полная свобода действий с поддержкой Drag-and-Drop для сортировки плиток.
  • Интуитивность: Современный интерфейс с плавной анимацией и обратной связью.
  • Умное подключение: Автоматический поиск сервера в локальной сети или ручной ввод IP с сохранением.

PC Сервер (Python)

  • Управление звуком: Быстрое переключение мута микрофона и системного звука (Speakers).
  • Интеграция с OBS: Дистанционный запуск и остановка записи через WebSocket.
  • Запуск софта: Возможность добавлять любые ярлыки (.exe, .lnk, .url) прямо из системного трея.
  • Удобство: Текущий IP-адрес сервера всегда виден в меню трея для быстрой настройки.

Структура проекта

Репозиторий разделен на две логические части:

  • /AndroidApp — Исходный код мобильного приложения (Kotlin/Android Studio).
  • /pc-server — Серверная часть на Python с использованием Flask и PyStray.

Технологический стек

Android: Kotlin, Retrofit2, SharedPreferences, ItemTouchHelper.

Backend: Python, Flask, OBS-WebSocket, PyCaw, PyStray, NirCmd.
